# TEAM_9
🍁 2023 단풍톤 9팀의 프론트엔드 레포지터리입니다.

## 프로젝트 활용방법
1. node.js를 설치해주세요.(npm도 같이 설치됩니다)
2. 콘솔에 필요한 라이브러리를 설치해주세요.
   2-1. `npm install` 
   2-2. `yarn install` 
3. 개발 서버 호스팅해주세요.
   3-1. `npm run dev` 
   3-2. `yarn dev` 
4. 프로젝트 빌드 방법입니다.
   4-1. `vite build` 
   4-2. `npm build` 
   4-3. `yarn build`
